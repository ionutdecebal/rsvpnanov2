# Instagram Speed Demo Notes

Temporary post concept saved for future reuse. This was designed as a short RSVP Nano social demo with the normal reader UI plus a header: `How fast can you read?`

## Timing

- Start at `50 WPM`.
- Increase by `50 WPM` per step.
- End at `1000 WPM`.
- Show the usual on-screen WPM readout.
- Stay at each WPM for two sentences.
- Keep the full demo around `45-60` seconds.
- Start word pacing generously at about `300 ms`.
- Tighten pacing as WPM rises, ending around `110 ms`.
- Keep punctuation delay above `100 ms`.
- Use very short early sentences and longer later sentences so each speed tier gets comparable screen time.
